# Property Message Templates

## Standard Listing (WhatsApp)

```
{Title}
{Location}
{Price} {Currency}
{Beds} bed | {Baths} bath | {Area}m2

{Description}

Ref: {Ref}
```

## Luxury Property

```
Exclusive: {Title}
{Location}

Price: {Price} {Currency}

{Detailed description highlighting premium features}

Features: Pool, Sea View, Private Parking
{Adjust to match actual features}

{Area}m2 | {Beds} Bedrooms | {Baths} Bathrooms

This is a rare opportunity in {area}. Would you like to arrange a private viewing?

Ref: {Ref}
```

## Batch Introduction (before sending multiple)

```
Hi {Name}!

I've found {count} properties that match what you're looking for. Let me share them with you — take a look and let me know which ones catch your eye!
```

## Batch Closing

```
Those are my top {count} picks for you!

Would you like to:
- Schedule viewings for any of these?
- See more options?
- Get more details on a specific one?

Just let me know!
```

## Email Subject Lines

- "Properties selected for you, {Name}"
- "{Count} properties matching your criteria — {Location}"
- "Your property shortlist from {Agency Name}"
- "New listings you might love — {Property Type} in {Location}"
